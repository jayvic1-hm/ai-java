# Simple Calculator (Java Swing)

간단하고 예쁜 GUI 계산기 프로그램입니다. Java Swing을 사용하여 제작되었으며, 더하기, 빼기, 곱하기, 나누기 기능을 지원합니다.

## ✨ 주요 기능

- **사칙연산** (+, -, *, /)
- **커스텀 계산기 아이콘** (BufferedImage로 직접 그린 아이콘)
- **연산 결과 다이얼로그** (성공 시 초록색 체크 아이콘 표시)
- **에러 처리** (0으로 나누기, 잘못된 입력)
- **예쁜 UI 디자인** (색상 커스터마이징, 라운드 처리)

## 🚀 설치 및 실행 방법

### 1. Java 개발 환경 준비
- JDK 8 이상 설치
- `javac`와 `java` 명령어가 터미널에서 실행 가능해야 함

### 2. 컴파일 및 실행

```bash
# 1. 컴파일
javac SimpleCalculator.java

# 2. 실행
java SimpleCalculator
```
